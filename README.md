# Başlarken

```
  $ npm install express-generator -g && npm install nodemon -g && npm install bower -g
  $ express crud --hbs --git
  $ cd crud && npm install
  $ nodemon bin/www
  $ bower init
  $ bower install bootstrap --save
```

Editör: Atom
ORM: Mongoose

## Notlar
  Bütün kurulumlara '--save' parametresi eklemelisin.
